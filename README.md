# VGC Colosseum API

JSON Server API for the VGC Colosseum application.

## Getting Started

### Prerequisites
- Node.js
- npm

### Installation

1. Clone this repository
```bash
git clone https://github.com/EricMartinez007/vgc-colosseum-api.git
cd vgc-colosseum-api
```

2. Install dependencies
```bash
npm install
```

3. Start the server
```bash
npm start
```

The API will run on `http://localhost:8088`

## Available Endpoints
